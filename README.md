# Projeto-NLP
Projeto com o objetivo de identificar se um texto foi escrito por um ser humano ou por uma inteligência artificial utilizando técnicas de Processamento de Linguagem Natural (NLP)


---

## **Descrição do Projeto**

O dataset possui duas colunas:
- Textos gerados por IA
- Textos gerados por humanos

Foi realizada uma análise exploratória e o pré-processamento dos dados para garantir a qualidade das informações usadas nos modelos.

### **Valores de Saída (Target)**
- `0`: Não gerado por IA
- `1`: Gerado por IA

---

## **Técnicas e Algoritmos Utilizados**

### **Modelo de Machine Learning**
- Regressão Logística

### **Pré-processamento**
- Limpeza de dados
- Vetorização
- Tokenização
